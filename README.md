# Flight Ticket Booking System ✈️

A backend console application built to manage flight reservations, passenger details, and ticketing operations. 

# Project Overview
This project is a functional Flight Ticket Booking System developed using *Python* and *MySQL*. It demonstrates the practical implementation of relational database management and backend business logic. The system allows users to seamlessly interact with a database to perform core reservation tasks.

# Tech Stack
* Language: Python 3
* Database: MySQL
* Libraries: mysql-connector-python

# Key Features
* Database Integration: Secure and stable connection between the Python application and the MySQL server.
* CRUD Operations: Capable of Creating, Reading, Updating, and Deleting booking records directly from the database.
* Interactive Console UI: Menu-driven interface for easy navigation and operation execution.
* Dynamic Querying:  Real-time data retrieval and updates using SQL queries embedded within Python functions.

# Learning Objectives
This project was built to strengthen foundational concepts in:
* Database Management Systems (DBMS) and Schema Design.
* Python-to-Database connectivity.
* Building scalable logic for real-world backend applications.
* Preparing for advanced AI and software engineering integrations.
